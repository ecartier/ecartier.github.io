Byzantine Fault Tolerant Machine Learning

In this master semester project, we attempt to test how different aggregation strategies are robust to various machine learning attacks (for example noisy data and gradients). Assuming a set of n workers, up to f of them being byzantine, we experiment different gradient descent update rules based on a combination of the vectors proposed by the workers. In particular, we test how the method developed in the lab (Krum) performs..

This project has been supervised by Peva Blanchard, Georgios Damaskinos, Rachid Guerraoui, El Mahdi El Mhamdi and Rhicheek Patra.
